## AIによる履歴メッセージ検索とは

MemFreeは、あなたの検索のタイトル、質問、および回答のためのAIセマンティック検索をサポートしています。

> これは有料機能ですが、すべてのユーザーが一時的に無料で体験できます。

### AIセマンティック検索とは

従来の全文検索やキーワード検索と比較して、AIセマンティック検索には以下の利点があります。

1. **多言語理解能力**

    - 使用する言語に関係なく、関連するコンテンツを見つけることができます。
    - 例えば：「AIセマンティック検索」と入力すると、「AI 智能搜索」に関連する結果も得られます。

2. **インテリジェントな誤り訂正**

    - 入力に誤りが含まれていても、正しい結果を見つけることができます。
    - 例えば：「MemFrre」と検索しても、「MemFree」に関連する結果が見つかります。

3. **セマンティック理解能力**

    - 単語間の関係を理解し、正確な一致に限定されません。
    - 例えば：「検索エンジン」を検索すると、「クエリツール」に関連するコンテンツも見つかります。

4. **コンテキストへの配慮**

    - 検索用語の意味を異なる文脈で理解し、文脈に基づいたより正確な検索結果を提供します。
    - 例えば、「Apple」は、果物やテクノロジー会社についての議論の区別を知的に行うことができます。
